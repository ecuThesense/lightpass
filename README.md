# Here is the project roadmap:
1. Abillity to import existing Database and create a new one.
2. See list os passwords and groups
3. Navigation via <-||-> and like VIM style
4. Abillity to create new groups and passwords + TOTP and save Database automaticaly
5. Abillity to copy data from an Entry
6. Abillity to export Database in several file formats like CSV, JSON, ENV, YAML
7. Abillity to store sensetive data such as ID, IBANS...
8. Abillity to store and simplyfy gpg keys usage
9. Abillity to change Database properties
10. Sorting and Searching though
11. Finding and deleting Dublicats
12. Abillity to Merge Databases
13. Abillity to restore deleted passwords
14. Support to create keyfiles
15. Fake Identity generation
16. Log storage
17. Policy based accsess

and much more..
